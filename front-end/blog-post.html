Spring Core - Part 1: Spring IoC, Inversion of Control in Spring.

1. What is IoC?

IoC ( I nversion o f C ontrol): Reverse control, it helps to change the control flow of the program flexibly.

Commonly used with Denpendency Injection.

You can review the article about Dependency Injection to better understand

2. Spring IoC

IoC Container is a component of implementing IoC.

In Spring, Spring Container (IoC Container) will create objects, assemble them together, configure objects and manage their lifecycle from creation to destruction. Spring containers use DI to manage components and objects to create an application. These components and objects are called Spring Bean (I will talk about Spring Bean in the following articles).

To create objects, configure, assemble them, Spring Container will read information from xml files and execute them.

IoC Container trong Spring có 2 kiểu là: BeanFactory and ApplicationContext
